## Notes

- Computed values (like `total`, `bonus`, `displayTotal`, `displayBonus` for characteristics) cannot be modified directly through Active Effects
- All attribute paths use the `system.` prefix
- Attribute types are either "Number", "String", or "Boolean" as indicated in the system
- Effects can stack multiple modifiers with proper priority handling
